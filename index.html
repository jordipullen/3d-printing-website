Assignment Details

The Assignment is to design a 3D-Printed case designed to protect an ESP32-C3 Zero microcontroller board during classroom use while additionally still allowing all the board features to be used.

ESP32-C3 Zero Protective Case Details

This website documents the design journey for a 3D printed protective case for the ESP32-C3 Zero microcontroller. The project includes Prototype 1, Prototype 2 and the Final Design.

The purpose of the website is to explain the problem, show how the case was improved through testing and feedback, and present the final design clearly.

Project Purpose

The ESP32-C3 Zero board is small and can be damaged during classroom use if it is dropped, handled roughly, or packed away without protection. The case is designed to protect the board while still allowing access to the USB port, pins and buttons.

The protective case needs to meet the following requirements:

  • Fit the ESP32-C3 Zero microcontroller board accurately.
  • Board needs to be protected during handling and classroom use.
  • The print must enable the board to be accessible for its USB port, pins and buttons.
  • The print must maintain stable and supported for the board.
  • Small vents or ventilation holes are to be added to enable airflow and basic cooling while in use and in storage.
  • The print is to be done using available classroom 3D printers and materials.
  • Has to be safe to use, practical, neat and professional in appearance.
